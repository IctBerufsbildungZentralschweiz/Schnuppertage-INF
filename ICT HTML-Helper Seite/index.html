Schritt 1: HTML Grundgerüst

Das ist das in der Präsentation erwähnte HTML Grundgerüst. Kopiere diesen Code in dein index.html file.

<!DOCTYPE html> <html lang="de"> <head> <meta charset="utf-8" /> <link rel="stylesheet" href="style.css"> <title>Meine erste Webseite</title> </head> <body> <p class="text">Test ob die Webseite funktioniert</p> </body> </html>

Kopiere diesen Code in dein style.css file.

.text { color: green; }

Speichere deine beiden Files und doppelklicke das index.html.
Wenn auf deiner Webseite der Text: "Test ob die Webseite funktioniert" in grün erscheint, hast du alles richtig gemacht!
Wenn nicht, dann frag mich oder schau in der Anleitung ob du nicht etwas falsch gemacht hast.

WICHTIG: Dein Code schreibst du nur innerhalb des öffnenden und des schliessenden Body-Tags.

Schritt 1: Erkärung

Das <!DOCTYPE html> sagt dem Browser, das es sich um ein HTML File handelt.
Die Sprache, die der Browser verwenden soll, wird sozusagen bekannt gegeben.

<html>: Das ist das öffende html-Tag. Findest du das schliessende?

Anschliessend folgt das <head> Tag. Hier sind Informationen vorhanden, welche der Browser braucht.
Z.B.: Die Codierung der Sprache.
Weiter folgt der Link zum CSS File und zum Schluss ist der Titel der Webseite. Den Text im Titel der Webseite
kannst du anpassen. Du findest den Titel als Text oben im Browser Tab.

Zuletzt folgt das <body>. Hierdrin befindet sich der Code.


CSS (Cascading Style Sheet)

Im Body-Tag deines HTML findest du ein p-Tag. Das p-Tag beinhaltet nur Text und bedeutet paragraph.
Wie du auch siehst, hat das p-tag ein Attribut welches class heisst.
Der Name der Klasse kannst du frei wählen. Achte darauf, dass der Name innerhalb der Hochkommas
und am selben Ort, wie im Beispiel steht.

Du greifst nun auf die Klasse "text" zu. Indem du .text {} im CSS File schreibst.
Nun kannst du diverse CSS Attribute wie "color" definieren.

Zum ausprobieren:

  • Ändere die Farbe des Textes
  • Verändere die Grösse oder die Schriftart

Änderungen musst du nur im CSS File machen. Der Text steht ja schon.

HTML Tabellen

HTML

<table> <tr> <th></th> <th></th> </tr> <tr> <td></td> <td></td> </tr> <tr> <td></td> <td></td> </tr> </table>

CSS

table { border-collapse: collapse; } td, th { border: 1px solid #dddddd; text-align: left; padding: 8px; }

Erklärung

Das <table> Tag beinhaltet die Tabelle mit ihren Zeilen und Spalten.

Das <tr> Tag heisst augeschrieben Table Row (Zeile).

Das <th> Tag heisst augeschrieben Table Header. Das ist der Titel der Spalte

Das <td> Tag heisst augeschrieben Table Data. Hier schreibst du den entsprechenden Wert welcher zum Titel gehört.
Achte auf die Rheinfolge, in welcher du die Tags schreibst.

Das CSS brauchst du nur in dein File zu kopieren. Falls du doch wissen möchtest, wie das genauer funktioniert frag mich :)
