Hier hast du Beispiele von CSS Befehlen

Vergiss nicht, die Klasse auf dem gewünschten Element zu definieren.

.text { color: #3A1DC8; font-size: 25px; font-family: arial; } .bild { heigth: 400px; width: 400px; } .hintergrund { background-color: black; } .weisserText { color: #ffff; }

Erklärung

Bei der .text{} Klasse setzt du die Farbe auf Blau (Du kannst die Farbe entweder auf englisch oder in Hexadezimal schreiben)
Hier eine Webseite, wo du Farben auswählen kannst und den Hexadezimalen Code wiederbekommst.

Hexadezimale Farben

font-size und font-family setzen die Grösse und die Schriftart.

Bei der .bild{} Klasse setzt du die Länge auf 200px und die Breite auf 300px. Probiere einfach mal ein bisschen aus.

Du kannst ebenfalls den Hintergrund eines Elementes ändern. Das machst du wie in der Klasse .hintergrund{}
